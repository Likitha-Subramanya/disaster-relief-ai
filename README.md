# AI Disaster Relief Coordination (Starter)

A modern React + Vite + TypeScript + Tailwind starter for an AI-powered disaster management & relief coordination app.

## Quick start

1. Install Node.js 18+.
2. Install deps:
   ```bash
   npm install
   ```
3. Run dev server:
   ```bash
   npm run dev
   ```
4. Open the printed local URL. You should see the landing with hero, features, map placeholder, report form, and relief centers.

## Tech stack

- React 18 + TypeScript
- Vite 5
- TailwindCSS 3
- Lucide icons
